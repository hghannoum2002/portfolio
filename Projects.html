SunAlarm

Designed a smart dawn simulator integrating multiple sensors (temperature, air quality, light, distance) and control modules (Bluetooth, touchscreen, real-time clock) to enhance the waking experience. The system drives RGB LEDs, an OLED display, and a buzzer, while a secondary board with ToF sensors monitors body position and transmits sleep data for analysis.

Ascon with fpga

This project implements secure electrocardiogram (ECG) analysis over a UART connection using ASCON-128 encryption. The repository includes an FPGA bitstream with its source files, a hardware architecture presentation, and a Python environment for ECG signal analysis (R-peak detection with Neurokit2 or PQRST analysis using SciPy).
